e-shell

e-shell e il terminale interattivo dell'ecosistema EOS. Unisce comandi di sistema e valutazione di espressioni E in un'unica interfaccia. Auto-detection del contesto: se il comando e un'espressione E valida, la valuta. Altrimenti lo esegue come comando di sistema.

Capabilities Features Integrazioni
Categoria
CLI / Terminale
Accesso
Closed source
Modalita
Interattiva + Script
Compatibilita
Linux, macOS, WSL

Funzionalita

  • Prompt interattivo: shell read-eval-print loop con colorazione e stato.
  • Auto-detection: riconosce se l'input e un'espressione E o un comando PATH.
  • Valutazione E: esegue espressioni matematiche, chiamate a funzione, assegnazioni.
  • Comandi built-in: cd, exit, navigazione filesystem.
  • History: buffer comandi con navigazione UP/DOWN.

Caratteristiche tecniche

  • Multi-piattaforma: pacchetto installabile via script.
  • Integrazione nativa con e-lang per esecuzione script .eee.
  • Colorazione output e prompt configurabile.
  • Modalita interattiva e scriptabile (pipeline).

Integrazioni

e-shell si integra con tutti i moduli EOS:

  • e-lang: parsing ed esecuzione di espressioni e script.
  • e-runtime: compilazione on-demand di moduli.
  • e-browser: avvio di sessioni browser da linea di comando.
  • EOS Core: connessione a kernel bare metal via seriale (futuro).

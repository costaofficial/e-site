e-browser

e-browser e un mini browser basato su WebView integrato nell'ecosistema EOS. Scritto in E, renderizza HTML/CSS/JS nativo via WebKit e puo essere controllato da script .eee per automazione browser.

Capabilities Features Integrazioni
Categoria
Browser / WebView
Accesso
Closed source
Engine
WebKit (wry)
Scritto in
E + Rust

Funzionalita

  • Navigazione web: barra URL, pulsanti Back/Forward/Reload.
  • Interfaccia E: definita in HTML/CSS/JS dentro un blocco :ui in file .eee.
  • Dark theme: tema Catppuccin Mocha integrato.
  • Controllo da script: avviabile e controllabile da altri moduli EOS.
  • Modalita kiosk: apertura diretta URL da linea di comando.

Caratteristiche tecniche

  • Basato su wry (WebKit WebView per Rust).
  • UI definita nel sorgente E come HTML inline.
  • Supporto per JavaScript e comunicazione con l'host E.
  • Auto-prefixing URL con https://.
  • Validazione URL e gestione errori di rete.

Integrazioni

e-browser si integra con l'ecosistema EOS per automazione e testing:

  • e-lang: script .eee possono aprire URL e interagire col browser.
  • e-shell: comando e --url https://... per apertura diretta.
  • e-runtime: compilazione del modulo browser come binario nativo.

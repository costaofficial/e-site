e-lang e il linguaggio open source al centro di EOS. Compila in LLVM IR e gira su bare metal. I moduli shell, browser, runtime e kernel completano la piattaforma.

Scopri e-lang EOS Core

Moduli della piattaforma

  • e-lang: Linguaggio open source. Sintassi, tipi, operatori, esempi — tutto pubblico e documentato.
  • e-shell: Terminale interattivo closed source. Auto-detection comandi E e PATH.
  • e-browser: Browser WebView closed source. Navigazione integrata nell'ecosistema.
  • e-runtime: Compilatore E closed source. Pipeline .eee → LLVM IR → codice nativo.
  • EOS Core: Kernel bare metal closed source. Scritto al 100% in E, zero C.

Quick start

Inizia con e-lang (open source)

Scarica l'eseguibile, crea il tuo primo script .eee e eseguilo:

curl -L https://github.com/costaofficial/e-lang/releases/latest/download/e -o e
chmod +x ./e
echo 'let x = 5
log x * 2' > hello.eee
./e run hello.eee
